## 📌 О проекте

Это веб-приложение представляет собой универсальный инструмент для организации работы и личных дел. Оно объединяет три ключевых модуля:

- **📋 Канбан-доска** — визуальное управление задачами с колонками: To Do, В работе, На проверке, Готово, Архив
- **🔗 Ссылки** — хранилище полезных URL-адресов с категориями и древовидной структурой
- **📝 Заметки** — быстрые текстовые записи для идей, планов на дейлики и важных мыслей

Все данные сохраняются в локальном хранилище вашего браузера (localStorage) и не требуют регистрации или интернета после первой загрузки.

## ✨ Возможности

### 📋 Канбан-доска
- **Drag & Drop** — перетаскивайте задачи между колонками
- **Фильтрация** — просматривайте задачи по статусу
- **Поиск** — быстрый поиск по названию и описанию
- **Приоритеты** — высокая, средняя, низкая важность
- **Дедлайны** — устанавливайте сроки выполнения
- **Категории** — группируйте задачи по темам (Работа, Личное, Учеба, Здоровье)
- **Ссылки** — прикрепляйте URL к задачам
- **Статистика** — общее количество, в работе, выполнено, архив
- **Экспорт в Excel** — выгружайте все задачи в таблицу

### 🔗 Ссылки
- **Древовидная структура** — создавайте вложенные категории
- **Карточки ссылок** — название, URL, описание
- **Фильтрация по категориям** — просматривайте ссылки в выбранной категории
- **Экспорт в Excel** — выгружайте ссылки с категориями

### 📝 Заметки
- **Карточки заметок** — заголовок и текстовое содержимое
- **Дата создания** — автоматическая метка времени
- **Поиск и фильтрация** (в планах)
- **Экспорт в Excel** — выгружайте все заметки

### 🎨 Оформление
- **Тёмная тема** — комфортная работа в тёмное время суток
- **Адаптивный дизайн** — корректно отображается на всех устройствах
- **Анимации** — плавные переходы и приятные визуальные эффекты
